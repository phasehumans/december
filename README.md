# December Monorepo

Autonomous AI Software Engineer workspace.

## Development Workflows

We use Turborepo to orchestrate builds, testing, and typechecking.

### Core Commands

- **Build all workspaces:**

    ```bash
    turbo run build
    ```

- **Typecheck all workspaces:**

    ```bash
    turbo run typecheck
    ```

- **Run all test suites:**

    ```bash
    turbo run test
    ```

- **Lint and format codebase:**
    ```bash
    bun run lint
    bun run format
    ```
