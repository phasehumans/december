# Security Policy

## Supported Versions

Currently, only the `main` branch of the repository receives security updates. If you are deploying this software, ensure you are running the latest commit from the `main` branch.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not disclose it publicly.

Instead, please send an email to the project maintainers directly at phasehumans@gmail.com or use GitHub's private vulnerability reporting feature. We will investigate all reports and work to provide a patch as quickly as possible.

Please include the following information in your report:

- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact or risk associated with the vulnerability.

We appreciate your effort in helping to keep this project secure.
