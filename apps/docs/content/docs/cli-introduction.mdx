The December CLI is a focused terminal-based workspace designed to bring the power of December's AI development pipelines directly into your local environment. It operates as a local agent, synchronizing local edits, orchestrating project sandboxes, and executing remote build commands.

## Key Capabilities

- **Local Workspace Sync**: Watch local files and synchronize code changes with the remote December compiler and visual editor in real-time.
- **Headless Build Pipeline**: Run tests, check TypeScript compiles, and run self-healing commands directly from your local terminal.
- **Secure Code Redemption**: Offline transaction-bound promotional credits can be securely generated and claimed via standard CLI commands.

## Installation

Install the December CLI globally via npm or bun:

```bash
# Using bun (recommended)
bun add -g @december/cli

# Using npm
npm install -g @december/cli
```

## Getting Started

Initialize your local workspace:

```bash
december init
```

This will authenticate your CLI session and link your current project folder to a December project space.
