A look under the hood at December's secure, lightning-fast compilation sandboxes.

Each workspace instance runs inside an isolated local container. We compile React 18 and strict TypeScript using custom, high-speed Vite bundlers.

| Metric               | Technology         | Performance Goal            |
| :------------------- | :----------------- | :-------------------------- |
| **Hot Reload**       | Vite HMR           | &lt; 150ms state updates    |
| **Server Execution** | Bun / Rust runtime | &lt; 3.0s total cold start  |
| **Database ORM**     | Prisma DB adapter  | &lt; 50ms schema migrations |
