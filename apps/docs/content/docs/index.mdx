## Core Capabilities

- **Web Workspace**: A visual-first design Canvas where you select elements, edit interfaces, and generate React components in real-time.
- **CLI Workspace**: A terminal-based developer tool designed for focused command-line workflows and code editing.
- **AI Code Agent**: An AST diff-based self-healing system that compiles, verifies, and publishes code automatically.
- **Edge Deployments**: Instant sub-second global CDN deployments under custom `december.dev` domains.
