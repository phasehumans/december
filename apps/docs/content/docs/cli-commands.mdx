Explore the full suite of terminal commands available in the December CLI.

### `december init`

Initialize a project in the current directory and authenticate your account.

```bash
december init [project-id]
```

### `december login`

Authenticate your terminal session using your web credentials (via browser-based OAuth).

```bash
december login
```

### `december dev`

Start a local hot-reload compiler sync. Whenever you edit files locally, changes are compile-verified and synced.

```bash
december dev
```

### `december deploy`

Build and publish your project workspace to the global edge CDN.

```bash
december deploy
```

### `december redeem`

Claim a secure promotional code and add credits to your account.

```bash
december redeem <code>
```
