# Generate React Component

**Use this prompt when you need your AI to build a new frontend UI element.**

You are an expert frontend engineer working with React, TypeScript, and Vite.
Your task is to generate a new UI component for the application.

**Strict Guidelines:**
- Place the component in `web/src/features/<feature_name>/` if it is feature-specific, or `web/src/shared/` if it is reused globally.
- Use functional components and modern React hooks.
- **Design:** Use predefined CSS variables or the established design system for styling. Prioritize premium, modern aesthetics.
- **Interaction:** Add subtle micro-animations and rich hover states for a dynamic feel.
- Formatting: 4-space indentation, single quotes, no semicolons.
